# Obsidian Shaders

**Obsidian Shaders** is a detailed Minecraft shaderpack for OptiFine/Iris with a cold, oppressive horror look. It is tuned for dark modpacks, cave exploration, monster-heavy worlds, abandoned structures, foggy forests, and survival horror maps.

## Features

- Five ready-to-use quality profiles: **WEAK**, **NORMAL**, **STRONG**, **VERY_STRONG**, and **ULTRA**.
- Shader Options menu groups for atmosphere, lighting, detail, post-processing, water, and advanced effects.
- Horror color grading with blue-black moonlight, low saturation, film grain, vignette, and subtle chromatic aberration.
- Configurable volumetric-style fog, dense night ambience, torch warmth, torch flicker, wet surface darkening, SSAO, bloom, and water darkness.
- Performance-focused toggles so the pack can be adjusted from low-end PCs to high-end systems.

## Installation

1. Copy this folder or a `.zip` made from it into your Minecraft `shaderpacks` directory.
2. Open Minecraft with OptiFine or Iris.
3. Go to **Options → Video Settings → Shaders** and select **Obsidian Shaders**.
4. Open **Shader Options** and choose one of the profiles or tune the grouped settings manually.

## Recommended horror setup

- Start with **NORMAL** or **STRONG**.
- Increase **HORROR_INTENSITY**, **FOG_DENSITY**, and **VIGNETTE_STRENGTH** for darker horror modpacks.
- Use **WEAK** for integrated graphics or heavy modpacks.
- Use **ULTRA** only with a strong GPU because it enables the highest shadow map size and strongest post-processing defaults.
